# Экранирование и доза — учебный веб-симулятор (Streamlit)

Назначение: визуализировать влияние расстояния и экранов на условную мощность дозы.
Модель учебная (относительные коэффициенты μ), типы излучения: гамма, бета, альфа, нейтроны.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate   # Win: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Что нового
- RU/EN локализация (переключатель в сайдбаре)
- Экспорт графика в PNG и данных в CSV
- Виды излучений с разными μ по материалам
- Исправлена ошибка сохранения сценариев (автосоздание папки scenarios)

## Новые возможности
- **Редактор μ**: страница `mu_editor` позволяет править учебные коэффициенты и сохранять в `data/mu_override.json`. Модель автоматически подхватывает их.
- **Таблица μ**: страница `mu_table` — быстрый просмотр и выгрузка CSV.
- **Экспорт/импорт сценариев**: страница `scenarios_io` — выгрузка JSON/ZIP, загрузка JSON/ZIP с выбором стратегии конфликтов.


## Экспорт пакета артефактов
Страница `export_bundle` формирует ZIP c PNG+CSV по выбранным сценариям (и сравнением при выборе 2 сценариев). Можно добавить таблицу μ и пользовательский `mu_override.json`.
